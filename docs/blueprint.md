 **App Name**: MediScan Assist

## Core Features:

- Prescription Scan: Scan medicine prescriptions using the device's camera to automatically extract relevant information.
- AI Dashboard Creation: Automatically generate a user-friendly dashboard based on scanned prescription data.
- Medication Status Tracking: Implement a checkbox system (Green - Taken, Yellow - Snoozed, Red - Missed) to track medication adherence.
- Dose Countdown: Provide a countdown timer for the next dose, visually displayed on the dashboard.
- Automated Voice Alerts and Pop-up Notifications: Send timely voice and pop-up reminders for upcoming doses to the user and notifications to the caregiver (e.g., 'Dad missed pills').
- Emergency Button: Include an emergency button for immediate access to ambulance services.
- Track Record and Prescription Updates: Maintain detailed medication track records, automatically updating the dashboard upon scanning updated prescriptions.

## Style Guidelines:

- Primary color: Soft blue (#89B7E5), conveying calmness and reliability.
- Background color: Very light blue (#EBF4FA), promoting a clean, uncluttered feel.
- Accent color: Pale violet (#D0B5E8), adding a gentle, reassuring highlight.
- Body and headline font: 'PT Sans' for a modern, readable feel.
- Use clear, intuitive icons for medication status and alerts.
- Dashboard layout should be clean and easy to navigate, with key information prominently displayed.
- Subtle transitions and animations to provide feedback on user interactions.
